Relearning the Foundations

Posted:

Today I decided to return to the fundamentals — Excel macros, VBA, lookups, and SQL. These are tools I’ve used before, but I’m sharpening them again because the truth is this: everything in tech comes back to the basics.

When I explained SQL to my friend at FedEx, I said: “Imagine walking up to the computer and asking it to pull every order they've touched in the last two years. Manually, you'd spend forever. SQL does it instantly.”

(This is Joe Kidd. The computer knows exactly what he touched.)

Relearning these skills is part of my path forward — toward stronger systems, cleaner automations, and better solutions for the clients I work with.
